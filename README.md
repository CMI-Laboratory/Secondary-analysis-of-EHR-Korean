# 전자의무기록 데이터의 이차분석

"전자의무기록 데이터의 이차분석"은 MIT Critical Data팀이 2016년도에 저술한 책인 “Secondary Analysis of Electronic Health Records”를 연세대학교 CMI lab (www.d-yoon.com) 에서 번역한 책입니다.

본 책에 사용된 코드와 데이터들이 여기에 정리되어 있습니다.

다운로드 받아 사용하시기 바랍니다.
